Requirements:

- UI/UX Designer
Job Specifications:
Create visually appealing and intuitive user interfaces for websites and web applications.
Conduct user research and gather feedback to inform design decisions and improve user experience.
Develop wireframes, prototypes, and mock-ups to effectively communicate design ideas.
Collaborate with development teams to ensure seamless integration of design elements.
Stay updated with the latest design trends, tools, and best practices.
- Front-End Developer
Develop responsive and interactive user interfaces using HTML, CSS, and JavaScript.
Collaborate with designers to implement visually appealing and user-friendly web designs.
Ensure cross-browser and cross-device compatibility of web applications.
Optimize website performance and loading speed.
Required Qualifications:
-Strong proficiency in HTML, CSS, and JavaScript.
-Experience with front-end frameworks such as React, Angular, or Vue.js.
-Familiarity with version control systems, such as Git.
-Understanding of responsive web design principles and mobile-first development.
Knowledge of front-end optimization techniques and performance measurement tools.
